O projeto Café de Angola é um site estático desenvolvido com HTML, CSS e JavaScript, que apresenta informações sobre cafés de Angola, seu histórico e produtos. Além do site, foi implementado um chatbot utilizando o Flowise, que permite aos usuários interagir e obter informações sobre o café ou tirar dúvidas.

Funcionamento do site
O site é responsivo, navegável e apresenta as seções principais: Home, Sobre o Café, Produtos e Contato. Todo o conteúdo está estruturado com HTML e estilizado com CSS, enquanto a interação dinâmica é feita com JavaScript."

O chatbot (Flowise)
O chatbot foi criado usando Flowise, que é uma ferramenta de criação de fluxos de conversa para inteligência artificial. Ele permite que o usuário faça perguntas sobre os tipos de café, métodos de preparo ou informações gerais sobre o tipo. O Flowise processa a entrada do usuário e retorna respostas baseadas em fluxo lógico definido (chat flow)."
